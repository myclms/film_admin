# FilmAdmin - Django

信息管理系统课程设计。基于django框架，前端使用jQuery和BootStrap。

一个个人电影收藏管理系统，记录自己的电影喜好。

## 主要功能

1. 用户模块
   - 注册/登录
   - 编辑用户信息
   - 修改密码
   - 注销

2. 电影信息管理模块
   - 展示用户所有电影/收藏夹/特定收藏夹内的电影
   - 增加电影
   - 删除电影/特定收藏夹内电影
   - 查询电影/特定收藏夹内电影
   - 修改电影
   - 增加收藏夹
   - 删除收藏夹
   - 查询收藏夹
   - 修改收藏夹

3. 统计模块
   - 统计总体用户信息 
   - 统计特定用户个人的信息
